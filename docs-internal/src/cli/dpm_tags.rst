dpm tags
--------

List published tags of an artifact

Synopsis
~~~~~~~~


List all tags associated with an artifact (dar/component) at an arbitrary OCI registry

::

  dpm tags [flags]

Examples
~~~~~~~~

::

  dpm tags oci://whatever.dev/bar/test

Options
~~~~~~~

::

  -h, --help                   help for tags
      --registry-auth string   path to a config file similar to docker’s config.json to use for authenticating to the OCI registry. Defaults to docker's config.json
