dpm add dar
-----------

add or update a dar in the project

Synopsis
~~~~~~~~


add or update a dar in the project

::

  dpm add dar <oci-uri> <--dependencies | --data-dependencies> [flags]

Options
~~~~~~~

::

      --data-dependencies   add the dar to the data-dependencies field
      --dependencies        add the dar to the dependencies field
  -h, --help                help for dar
      --insecure            use http instead of https for OCI registry
